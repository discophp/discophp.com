Install

Using Composer

First you will want to clone Disco from github

git clone https://github.com/jbhamilton/disco.git

If you don't have composer installed globally grab a copy into the project directory

curl -sS https://getcomposer.org/installer | php

Put the composer.phar in the projects root directory

Now install Disco!

php composer.phar install

Congrats now Disco is installed!
