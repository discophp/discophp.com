Crypt Facade

The Crypt Facade provides functionality for AES encryption using a 32bit key size through the mcrypt PHP module, and hashing using the sha512 algorithmn

You must configure your AES_KEY256 setting variable in .env.*.json to a 32 bit random key for your encryption to be secure

You must configure your SHA512_SALT_LEAD and SHA512_SALT_TAIL setting variable in .env.*.json to a random key of any length for your hasing to be secure

Example Usage

Encryption

    $crypt = Crypt::encrypt('encrypt this string!');

Decryption

    $decrypt = Crypt::decrypt($encryptedString);

Hashing

    $hashedPassWord = Crypt::hash('hashthispassword');
