Configuration

There are 3 primary configuration files for Disco

  • .config.php - Your local environments settings
  • .dev.config.php- Your development environment settings. Will only attempt to be loaded if app is in DEV MODE ( APP_MODE=DEV ). Any variables you have set in your .dev.config.php will override its production .config.php counterpart value. The .dev.config.php does not have to exist or be used while in DEV MODE
  • .mail.conf.php - Your Email settings

.config.php - Your applications primary configuration file

Most of these should be pretty self-explanatory.

  • "URL" - The URL of your project ( somesite.com || sub.localhost )
  • "APP_MODE" - The mode your application is running ( DEV || PROD )
  • "COMPOSER_PATH" - Installation directory of Composer
  • "DB_*" - Set up your DB config options for MySQL.
  • "MEMCACHE_*" - Set up your Memcache Host and Port
  • "AES_KEY256" - This will be set to a 32 length value for you after install
  • "SHA512_SALT_*" - This will be set to a 12 length value for you after install

About the variable APP_MODE

This controls what mode your application is running in.

You can run in two different modes; DEV or PROD

When you are in DEV mode your application will do these things differently:

  • Send all emails to the email address set in the .mail.config.php file with the key DEV_MODE_SEND_TO
  • If .dev.config.php exists, override the .config.php values with any of its values

You can set any variables in your .config.php files you want, they will be accessible through out your application through the PHP super global $_SERVER

.mail.config.php - Your email configuration file

Set up your SMTP, TLS, and/or SSL email servers settings.

Set up your email addresses by replacing johns-email with your defined key and and appropriate information.

You can add as many email addresses as you want to the .mail.config.php file.

Configuring your Apache2 or Nginx Server to handle URL rewriting

For Apache2 Servers

Make sure you have the rewrite module enabled

sudo a2enmod rewrite

.htaccess rules


    <IFModule mod_rewrite.c>

        RewriteEngine On

        RewriteCond %{REQUEST_FILENAME} !-d
        RewriteCond %{REQUEST_FILENAME} !-f
        RewriteRule ^ index.php [L]

    </IfModule>

For Nginx Servers

Add this directive to your sites configuration file


    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }
