Configuration

There are 3 primary configuration files for Disco

  • .env.[Your Machine Name].json - Your local environments settings
  • .env.json - Your production environments settings
  • .mail.settings.json - Your Email settings

.env.[Your Machine Name].json

Rename the file .env.local.json that comes in the standard distribution to .env.[Your Machine Hostname].json

If you don't know your machines hostname run the command

hostname

Most of these should be pretty self-explanatory.

  • "URL" - The URL of your project
  • "COMPOSER_PATH" - Installation directory of Composer
  • "DB_*" - Set up your DB config options for MySQL.
  • "MEMCACHE_*" - Set up your Memcache Host and Port
  • "AES_KEY256" - Set to a random 32 length string
  • "SHA512_SALT_*" - Salt your hash values

You can set any variables in your .env files you want, they will be accessible through out your application through the PHP super globals $_SERVER and $_ENV

.env.json

This file is the same as .env.local.json but for your production environment

It will be used when the machine name doesn't match a settings file

So to use your production settings just make sure your local environment machines hostname doesn't match your production servers

.mail.settings.json

Set up your SMTP, TLS, and/or SSL email servers settings

Replace email@site.com (and the password obviously) with an active email address associated with one of your servers

You can add as many email addresses as you want to the json file
